16 tiết học thực chiến, hoàn thành project có thể deploy

Cài đặt Claude Code trên macOS/Windows/Linux, cấu hình API key, làm quen giao diện dòng lệnh, và thực hành câu lệnh đầu tiên.

Học cách cấu trúc prompt để nhận được code đúng ý, xử lý các tình huống AI tạo code sai, và tối ưu hóa context window.

Sử dụng AI để phân tích yêu cầu nghiệp vụ, thiết kế database schema, vẽ sơ đồ kiến trúc, và lên kế hoạch phát triển. Thực hành trên ProjectOS.

Xây dựng complete backend: thiết lập Firebase project, viết authentication flow, thiết kế Firestore database, cấu hình security rules, và viết Cloud Functions.

Thiết kế và xây dựng giao diện người dùng với Tailwind CSS, kết nối Firebase realtime data, xử lý các nghiệp vụ CRUD, form validation, và state management.

Sử dụng Claude Code để phân tích bug, refactor code sạch hơn, viết unit test tự động, và cải thiện performance.

Bổ sung các tính năng nâng cao: biểu đồ với Recharts, thông báo real-time, upload file lên Firebase Storage, và collaborative features.

Deploy hoàn chỉnh lên Vercel, thiết lập CI/CD tự động, cấu hình custom domain, tối ưu SEO, và monitoring với Vercel Analytics.
